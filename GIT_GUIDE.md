# คู่มือการอัพเดท Code ขึ้น GitHub

เมื่อคุณมีการแก้ไขโค้ดและต้องการอัพเดทขึ้น GitHub ให้ทำตาม 3 ขั้นตอนง่ายๆ นี้ใน Terminal:

## 1. เลือกไฟล์ที่จะอัพเดท (Stage)
คำสั่งนี้จะบอก Git ว่าเราต้องการบันทึกการเปลี่ยนแปลงของไฟล์ทั้งหมด
```bash
git add .
```

## 2. บันทึกการเปลี่ยนแปลง (Commit)
คำสั่งนี้จะบันทึกไฟล์ที่เลือกไว้ลงในประวัติ พร้อมข้อความกันลืมว่าแก้อะไรไปบ้าง
```bash
git commit -m "ข้อความอธิบายว่าแก้อะไรไป เช่น แก้สีปุ่ม, เพิ่มตาราง"
```

## 3. ส่งขึ้น GitHub (Push)
คำสั่งนี้จะส่งไฟล์ที่บันทึกไว้ขึ้นไปที่ GitHub
```bash
git push
```

---
**สรุปสั้นๆ:**
1. `git add .`
2. `git commit -m "message"`
3. `git push`
