# วิธีรันเว็บโดยไม่ใช้ Go Live

เนื่องจากคุณได้ติดตั้ง **Node.js** แล้ว คุณสามารถจำลอง Server ได้ง่ายๆ ด้วยคำสั่งนี้:

## 1. เปิด Terminal ใหม่
**สำคัญ:** เนื่องจากเพิ่งติดตั้งโปรแกรมไป คุณต้องปิดหน้าต่าง Terminal เดิมแล้วเปิดใหม่ (หรือปิด VS Code แล้วเปิดใหม่) เพื่อให้เครื่องรู้จักคำสั่ง `npx`

## 2. พิมพ์คำสั่ง
```bash
npx http-server
```
(ถ้ามันถามว่าให้ติดตั้งไหม ให้ตอบ `y` แล้วกด Enter)

## 3. เปิดเว็บ
เมื่อรันเสร็จ มันจะโชว์ลิงก์มาให้ (เช่น `http://127.0.0.1:8080`)
ให้กดปุ่ม **Ctrl + คลิก** ที่ลิงก์นั้นเพื่อเปิดเว็บได้เลย!

---
**วิธีลัด:**
ถ้าไม่อยากพิมพ์ยาวๆ ครั้งหน้าพิมพ์แค่:
```bash
npx http-server
```
จบเลย!

---
## (ทางเลือก) สำหรับคนชอบ Python
ถ้าคุณถนัด Python มากกว่า ก็ใช้คำสั่งนี้แทนได้ครับ (ผลลัพธ์เหมือนกัน):
```bash
python -m http.server
```
แล้วเข้าเว็บที่ `http://localhost:8000` ครับ
